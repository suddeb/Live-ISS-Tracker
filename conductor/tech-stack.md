# Tech Stack

## Programming Language
- **JavaScript (Node.js):** Primary language for both frontend and backend development.

## Backend Frameworks
- **Express:** Web server framework for handling API routes and static file serving.
- **Socket.IO:** Real-time bi-directional communication for pushing position updates.

## Frontend Libraries
- **Leaflet.js:** Open-source JavaScript library for interactive maps.
- **Socket.IO Client:** Integration for receiving real-time server updates.

## Key Dependencies
- **Axios:** HTTP client for external API requests (e.g., wheretheiss.at, corquaid.github.io).
- **Satellite.js:** SGP4/SDP4 propagation for orbital path calculations.
- **Node-cron:** Task scheduling for periodic updates.
- **Dotenv:** Environment variable management.

## Deployment & Hosting
- **Node.js Environment:** Target version >= 16.0.0.
- **Client-Side:** Pure HTML/CSS/JS (no heavy framework like React/Vue for simplicity and performance).
